关于点击

浙江点击资产管理有限公司为和讯网的官方唯一指定合作单位，目前主要从事和讯网旗下私募产品的发行和销售、财富管理业务的承销、培训业务、广告代理、金融衍生品等产品的投资咨询和销售。

我们的团队

总裁，曾为天津贵金属交易所名牌会员、无锡君泰贵金属交易中心创始终身荣誉会员，曾在无锡君泰贵金属交易中心任职期间，创造了会员单位提供了公司全体业务总量50%以上的业内神话。在市场开发及运营管理上具有较强的实战经验和科学理念，在行业内具有强大的市场影响力和感召力，并具有较强的社会责任感，获得了良好的市场口碑和认同度。

赵智兵

行政副总裁，浙江大学金融学在读硕士，2007年至2012年就职于平安银行杭州分行公司业务管理部，2013年至今就任于杭州红松资本管理有限公司董事长。

公司新闻

浙江点击携手和讯网开启全方位战略合作 2015年12月1日，浙江点击资产管理有限公司与和讯信息科技有限公司正式签订战略合作协议，这也意味着浙江点击金融+互联网+媒体的框架布局已初步完成。 浙江点击以全国独家总代的身份正式开启和讯网的资源整合和合作。和讯网作为国内第一的专业财经门户网站，经过多年的专注耕耘，确立了自己在业内的优势地位和品牌影响力，成为财经互联网专业、高端及品质的代表，在广大用户中拥有极强的专业口碑。浙江点击同样拥有专业化的项目投研团队、风控管理团队、成熟的市场运作经验、专业化的分销体系等优势。此次合作意味着两家公司今后在资管产品开发、销售；互联网广告业务；金融培训；私募产品管理；投资咨询业务等各领域的战略布局已经正式启动。
浙江点击助力天天快递踏上资本之路 2016年2月25日，浙江点击资产管理公司作为资方受邀参加天天快递行业研讨会暨上市发布会。公司董事长赵智兵，副总裁钱江鹏、祝渊，和讯金融衍生品事业部总经理邵欣伟作为嘉宾出席本次会议。 当日天天快递公司宣布完成6亿元A轮融资，由中金前海发展（深圳）基金管理有限公司领投，浙江点击资产管理有限公司、贵州盘江资产管理有限公司、青岛海尔创业投资有限责任公司、浙江承象投资管理有限公司、大航海资本等5家投资机构跟投。天天快递此次所获资金将主要用于对配送网及分拨中心的建设、对加盟网点的补助以及对信息系统的建设和升级。随着此次融资的完成，天天快递上市工作也正式宣布启动。 更值得注意的是，建设云仓也是天天快递此次所融资金的重点用途。当天，天天快递发布了旗下一站式供应链互联仓储平台“能容云仓”，其以“资本+技术+管理+资源”搭建平台，组织并优化仓库、仓储配套服务商、快递分拨中心和网点资源共享与信息互通，以满足不同细分市场的需求。 浙江点击也是看中天天快递近几年的高速成长性及未来巨大的市场潜力，积极参与了本次项目的整体运作。作为主要资方，不但为天天快递提供了资本助力，更将整合自身的优势资源，为企业提供媒体支持，专业渠道等全方位的服务，协助企业更快速更稳健的发展。
